# 3 курс — 1 семестр

!!! note "Розділ у роботі"
    Конспекти дисциплін цього семестру додаватимуться по мірі їх вивчення.

## Про що цей розділ

Дисципліни **1 семестру 3 курсу** освітньо-професійної програми «Право інформаційної безпеки» — бакалавр заочна (3 курс).

## Що тут з'явиться

- Перелік дисциплін семестру за робочим навчальним планом.
- Конспекти лекцій по кожній дисципліні.
- Матеріали практичних / семінарських занять.
- Підготовка до сесії: контрольні питання, форми звітності.
- Курсові роботи (де передбачено).
