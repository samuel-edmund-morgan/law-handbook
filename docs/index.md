# Юридичний компендіум

!!! abstract "Що це"
    **Особистий конспект** Samuel E. Morgan, побудований навколо однієї цілі — здобути спеціальність **D8 «Право»** в **Національній академії Служби безпеки України** і вести по ній якісні нотатки протягом усіх 7 курсів навчання та аспірантури.

    Сайт обслуговує **дві фази** одного маршруту:

    1. **Підготовка до вступу** (травень-серпень 2026): 4 індивідуальні усні співбесіди замість НМТ, ціль — конкурсний бал ≥ 170 (мінімум прохідний — 150).
    2. **Конспект навчання** (з вересня 2026): бакалавр заочна (1-5 курс, ОПП «Право інформаційної безпеки») → магістр заочна (6-7 курс, ОПП «Інформаційне право») → аспірантура.

## Чому існує

- **Особиста мотивація**: друга вища на основі магістра з кібербезпеки (НРК7), кар'єрна траєкторія в перетині права, інформаційної безпеки і державної безпеки.
- **Технічна мотивація**: повторне використання знайомого MkDocs Material стека (як у [handbook.morgan-dev.com](https://handbook.morgan-dev.com)) під новий домен і нову предметну область.
- **Системна мотивація**: один зріз правди про те, що читалось/розумілось — без розпорошення між нотатками в Notion, Apple Notes, паперовим конспектом і папкою на iCloud. Усе тут.

## Бренд і дизайн

- **Логотип** — стилізована **Феміда**: пов'язка на очах (безсторонність), розкинуті руки тримають ваги (зважування доказів). Для favicon — спрощені ваги.
- **Палітра** — slate (тёмна тема), чорний primary, червоний accent. Шрифти Fira Code / Fira Mono.
- **Мова** сайту — повністю українська. Перекладу немає.
- **Навігація**: верхній бар — основні розділи (Головна / Підготовка до вступу / Курси / Практика); лівий сайдбар — ієрархія поточного розділу; правий — зміст поточної сторінки.
