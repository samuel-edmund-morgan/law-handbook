# Практика

!!! note "Розділ у роботі"
    Матеріали практики додаються по мірі її проходження.

## Про що цей розділ

Тут зібрано матеріали з усіх видів **практики під час навчання**:

- **Навчальна практика** на молодших курсах бакалаврату (1-2 курс).
- **Виробнича практика** на старших курсах бакалаврату (3-5 курс).
- **Переддипломна практика** перед державною атестацією (5 курс).
- **Магістерська практика** (6-7 курс, ОПП «Інформаційне право»).
- **Дослідницька практика** в аспірантурі.

## Що зберігається по кожній практиці

- Наказ Академії про направлення на практику.
- База практики (організація, відділ, керівник).
- Програма практики (затверджена кафедрою).
- Щоденник практики (структуровано по днях / тижнях).
- Звіт з практики (повний текст + матеріали захисту).
- Відгук керівника практики від бази.
- Оцінка та залік.
