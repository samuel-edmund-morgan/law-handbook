# Формули розрахунку балів

!!! info "Заглушка"
    Сторінка створена як частина каркасу. Контент наповнюється поетапно протягом підготовки.

## Призначення сторінки

Конкурсний бал = сума по предметах × коефіцієнт. Українська 0.45 + математика 0.4 + історія 0.4 + англійська 0.5. Прохідний 150, цільовий 170+, шкала кожного предмета 100-200.
