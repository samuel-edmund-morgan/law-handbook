# Family. Family relationships

!!! note "Розділ у роботі"
    Конспект цього розділу ще наповнюється. Поки тут — короткий опис того, що буде.

## Про що цей розділ

Склад родини, ролі, стосунки між поколіннями, конфлікти й примирення. Vocab: relatives, in-laws, upbringing.

## Що тут з'явиться

- Опорна структура матеріалу
- Типові питання білета
- Опорні тези відповіді
- Поширені помилки й пастки
- Зв'язки з іншими розділами
