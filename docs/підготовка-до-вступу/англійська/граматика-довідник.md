# Граматика-довідник

!!! note "Розділ у роботі"
    Конспект цього розділу ще наповнюється. Поки тут — короткий опис того, що буде.

## Про що цей розділ

Стиснутий граматичний референс під B2: часи (12 active + passive), conditionals (0/1/2/3/mixed), modals, reported speech, articles, comparatives/superlatives, gerunds vs infinitives, complex sentences, common pitfalls.

## Що тут з'явиться

- Опорна структура матеріалу
- Типові питання білета
- Опорні тези відповіді
- Поширені помилки й пастки
- Зв'язки з іншими розділами
