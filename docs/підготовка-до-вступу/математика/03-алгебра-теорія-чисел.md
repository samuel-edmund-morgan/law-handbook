# 03. Алгебра та теорія чисел

## Про що цей розділ

Лінійна алгебра: векторні простори, СЛР, матриці, визначники, лінійні оператори. Многочлени, НСД/НСК, корені; прості/складені числа, решето Ератосфена, конгруенції, теореми Ейлера-Ферма.

## Що тут з'явиться

- Опорна структура матеріалу
- Типові питання білета
- Опорні тези відповіді
- Поширені помилки й пастки
- Зв'язки з іншими розділами
