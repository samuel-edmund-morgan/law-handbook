## Призначення сторінки

5 розділів програми: алгебра/початки аналізу, планіметрія/стереометрія, алгебра/теорія чисел, матаналіз, геометрія. Білет = 2 теорії + 2 задачі. 60 хв підготовки. Шкала: 7-8 → 161-174, 9-10 → 175-189, 11-12 → 190-200.
