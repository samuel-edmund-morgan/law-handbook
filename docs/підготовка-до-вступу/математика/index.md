# Математика — огляд

!!! note "Розділ у роботі"
    Конспект цього розділу ще наповнюється. Поки тут — короткий опис того, що буде.

## Про що цей розділ

5 розділів програми: алгебра/початки аналізу, планіметрія/стереометрія, алгебра/теорія чисел, матаналіз, геометрія. Білет = 2 теорії + 2 задачі. 60 хв підготовки. Шкала: 7-8 → 161-174, 9-10 → 175-189, 11-12 → 190-200.

## Що тут з'явиться

- Опорна структура матеріалу
- Типові питання білета
- Опорні тези відповіді
- Поширені помилки й пастки
- Зв'язки з іншими розділами
