# Українська мова — огляд

!!! note "Розділ у роботі"
    Конспект цього розділу ще наповнюється. Поки тут — короткий опис того, що буде.

## Про що цей розділ

7 розділів: фонетика/орфоепія/орфографія, лексикологія/фразеологія, словотвір, морфологія, синтаксис, стилістика, розвиток мовлення. Білет = 2 питання, відповідь 20-25 речень, 3 хв підготовки. Шкала лінійна: 8 → 170, 12 → 200.

## Що тут з'явиться

- Опорна структура матеріалу
- Типові питання білета
- Опорні тези відповіді
- Поширені помилки й пастки
- Зв'язки з іншими розділами
