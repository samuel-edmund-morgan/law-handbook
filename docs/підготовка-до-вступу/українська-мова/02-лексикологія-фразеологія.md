# 02. Лексикологія. Фразеологія

!!! note "Розділ у роботі"
    Конспект цього розділу ще наповнюється. Поки тут — короткий опис того, що буде.

## Про що цей розділ

Лексичне значення; багатозначні/однозначні; переносні значення; омоніми/синоніми/антоніми/пароніми; питома й запозичена лексика; стилі лексики; терміни, неологізми, історизми; фразеологізми.

## Що тут з'явиться

- Опорна структура матеріалу
- Типові питання білета
- Опорні тези відповіді
- Поширені помилки й пастки
- Зв'язки з іншими розділами
