## Принципи ритму

!!! tip "Що працює на заочному форматі другої вищої"
    1. **Маленькі щоденні дози** > вихідних запоїв. 30-60 хв якісної концентрації щодня.
    2. **Обертання предметів** у тижні: 2 «теплих» (укр + історія) + 1 «холодний» (математика).
    3. **Активне відтворення**, не пасивне читання: після кожного блоку — спроба відповіді на 1 питання вголос на 20-25 речень, без шпаргалки.
    4. **Перевірка по чек-листах** на дашборді раз на тиждень (неділя ввечері).
    5. **Резерв 1 буферний день** на тиждень для непередбачуваного — без планування.

## Що НЕ робити

!!! warning "Антипатерни"
    - Зубрити всі 32 блоки історії дослівно — програма дає теми, а не текст підручника.
    - Конспектувати математику без реш. задач — без практики бал ≤ 5/12.
    - Готувати англійську по списках слів без mononlogues — на співбесіді треба говорити, а не пасивно знати.
    - Залишати «розвиток мовлення» (укр) на останній день — це поверхнева тема, але дає легкі +1 бал.
